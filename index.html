Overview

Kagibox K.B. II is a high-performance cryptographic key generation node engineered by Matouš Jírovec. Drawing heavy aesthetic inspiration from the iconic industrial design language of Teenage Engineering, it features a tactile, analog-inspired interface that delivers mathematically secure entropy through native browser APIs.

Operation Guide
[ PWR.SYS ] Power Toggle I / O
Flip physical switch to I to boot and run system check. Flip to O to zeroize display cells and pop physical keys.
[ OP.MODE ] Mode Selection M
Cycle generator modes with key or select on rocker panel:
▪ Random: Custom character pools. ▪ Phonetic: Memorable BIP-39 lists. ▪ Numeric: Optimized PIN code output.
[ VAL.SLDR ] Tactile Fader Slider
Slide fader along linear track to adjust target length (characters, memorized word blocks, or PIN digit sizes).
[ GEN.COMP ] Compute Keys Space
Press key or click `RUN GENERATION` to process raw entropy via mathematically unbiased rejection sampling.
[ OUT.SYNC ] Clipboard Copy C / Display
Tap VFD display glass or press key to copy key. Ambient status beacon flashes in rapid data transmission sequence.
[ MEM.ZERO ] Purge Buffer P
Press key or click `PURGE BUFFER` to flush memory. Zeroizes variables and sweeps the OS clipboard clean.
[ PWR.GRID ] Battery Mode USB-C
Unplug USB-C cable to engage battery cells. Activates dimming power save and 15s deep sleep auto-timeout.
[ LED.TELM ] Strength Pulse LED
Side LED beacon interval signals entropy levels: ranges from slow warning flash (3.2s) to secure pulse (0.4s).
Security Architecture
[ CRYP.RNG ] Entropy Node
Utilizes standard 256-bit mathematical randomness gathered via browser-level crypto.getRandomValues() API, fully bypassing predictable linear seeds.
[ MATH.MOD ] Unbiased Modulo
Implements rejection sampling to discard upper remainder overflow ranges. Eliminates modulo bias for mathematically uniform distribution.
[ MEM.ZERO ] Local Wiping
Generated keys reside only in ephemeral JS memory. Calling the purge sequence writes empty strings to variables and zeroizes the host clipboard.
[ CSP.SAND ] CSP Sandboxing
Strict meta headers isolate execution. Disables external stylesheet calls, inline script bindings, tracking pixels, and unauthorized server hooks.
[ DEV.ISOL ] Client Shielding
Excludes key elements from device dictionary checks (spellcheck="false") and automated translation blocks to block outbound telemetry.
